Project Problem Statement

Problem Statement

University students often struggle to manage their monthly finances. Without a proper record, it is difficult to track where money is being spent, leading to overspending and a lack of savings. Manual tracking on paper is tedious and prone to loss.

Scope of the Project

This project aims to develop a digital solution to record and analyze personal financial data.

It allows users to input daily expenses easily.

It provides summaries to help users understand their spending habits.

It stores data permanently so it can be accessed anytime.

Target Users

Students living in hostels or PG accommodations.

Anyone looking for a simple, offline tool to track daily spending.

High-Level Features

User Input: Simple interface to add cost, date, and type of expense.

Storage: Uses a CSV file to mimic a database for data storage.

Reporting: Generates a "Total Spent" summary and specific category totals.

Error Handling: Robust validation to ensure data integrity (e.g., no text in amount fields).
